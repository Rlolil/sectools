Windows üçün Təhlükəsizlik Alətləri

Windows üçün ən yaxşı kibertəhlükəsizlik alətlərinin geniş kolleksiyası. Təhlükəsiz yükləmə linkləri və VirusTotal yoxlamaları ilə.

Siyahıya Keçid

Kali Linux Quraşdırması

Penetrasiya testləri üçün ən populyar Linux distribusiyası

1ISO Yükləmə

Kali Linux rəsmi saytından ən son ISO faylını yükləyin:

https://www.kali.org/get-kali/

• Kali Linux 64-bit ISO seçin

• Ölçü: ~3.5GB

2Virtual Machine Yaratma

VirtualBox və ya VMware-də yeni VM yaradın:

Tövsiyə olunan ayarlar:

• RAM: 4GB (minimum 2GB)

• HDD: 40GB

• CPU: 2 core

• Network: NAT

3Quraşdırma Prosesi

ISO faylını mount edib quraşdırmaya başlayın:

1. Graphical Install seçin

2. Dil və region seçin

3. Disk partitioning (Guided - use entire disk)

4. İstifadəçi və parol təyin edin

4İlk Konfiqurasiya

Quraşdırmadan sonra əsas ayarları edin:

# Sistem yeniləmə

sudo apt update && sudo apt upgrade -y


# Əlavə alətlər

sudo apt install -y kali-linux-large

5Faydalı Məsləhətlər

⚠️ Təhlükəsizlik:

• Root istifadəçisi ilə işləməyin

• SSH-ı deaktiv edin

• Firewall konfiqurasiya edin (UFW)

• Mütəmadi snapshot yaradın

VirtualBox Quraşdırması

Pulsuz və açıq mənbə virtualizasiya platforması

1VirtualBox Yükləmə

Oracle VirtualBox rəsmi saytından yükləyin:

https://www.virtualbox.org/wiki/Downloads

• Windows hosts seçin

• Pulsuz və açıq mənbə

2Quraşdırma

Administrator hüquqları ilə quraşdırın:

Quraşdırma addımları:

1. .exe faylını işə salın

2. Default ayarları saxlayın

3. Network interface xəbərdarlığını qəbul edin

3Extension Pack

Əlavə funksiyalar üçün Extension Pack yükləyin:

Əlavə funksiyalar:

• USB 2.0/3.0 dəstəyi

• RDP dəstəyi

• Disk şifrələmə

4Performans Optimallaşdırması

VM performansını artırmaq üçün:

Settings → System:

• Processor: 2+ CPU

• Enable VT-x/AMD-V

• Video Memory: 128MB

VMware Quraşdırması

Professional virtualizasiya həlli

1VMware Yükləmə

VMware rəsmi saytından yükləyin:

https://www.vmware.com/products/workstation-player.html

• Workstation Player (pulsuz)

• Workstation Pro (ödənişli)

2Quraşdırma Prosesi

Administrator hüquqları ilə quraşdırın:

Quraşdırma seçimləri:

1. Enhanced Keyboard Driver

2. Add VMware to PATH

3. Check for updates on startup

3VMware Tools

Performans üçün VMware Tools quraşdırın:

Faydaları:

• Daha yaxşı performans

• Clipboard paylaşımı

• Drag & drop, Auto screen resize

4Snapshot İdarəetməsi

Snapshot strategiyası (Pro):

• Clean install snapshot

• Pre-test snapshot

• VM → Snapshot → Take Snapshot
